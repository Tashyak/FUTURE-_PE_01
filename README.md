# AI Website Copy Prompt System for Local Businesses

## Project Overview

This project demonstrates how **Prompt Engineering and AI tools** can be used to generate **conversion-focused website copy for local businesses**.

Many small businesses lose potential customers because their website content is unclear, generic, or lacks strong calls-to-action. Hiring professional copywriters can be expensive, so this project creates a **structured prompt system** that generates high-quality website copy using AI.

The prompts are reusable and can work for multiple types of businesses such as:

* Cafes
* Salons
* Clinics
* Coaching institutes
* Agencies and freelancers

The generated content is designed to be **clear, persuasive, and ready for real websites**.
